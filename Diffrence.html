What's the Diffrence?

HTML

Hypertext Markup Language

Create the structure

  • Controls the layout of the content
  • Provides structure for the webpage design
  • The fundamentals building block for any webpage

CSS

Cascading Style Sheet

Stylize the website

  • Applies style to the webpage elements
  • Targets various screen sizes to make webpages responsive
  • Primarily handles the "look and feel" of a webpage

Javascript

Increase interactivity

  • Adds interactivity to a webpage
  • Handles complex functions and features
  • programmatic code which enhaces functions
